StoryMaker

Welcome to StoryMaker, a tool to make it more fun to get started on your firts script for the cinemas or the theatre.

It's easy to get started, just click the icon to create your script. Or click the same icon when you want to create a new act within that script.

But for each new script you create we have already created the three acts that you will need: 1. The Beginning, 2. The Middle and the 3. The end

But create how many scenes you like. Just make em good.

And remember, drama is conflict.
